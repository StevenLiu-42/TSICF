📦 安裝命令
─────────────────────────────────────────────────────────────────
npm install                          # 安裝所有依賴
npm install axios bootstrap gh-pages tailwindcss --save  # 安裝特定包

🚀 運行命令
─────────────────────────────────────────────────────────────────
npm start                           # 本地開發服務器 (localhost:3000)
npm test                            # 運行測試
npm run build                       # 建立生產版本
npm run deploy                      # 部署到 GitHub Pages

src/utils/likeManager.js            # ❤️ 點讚管理

🔧 Google Apps Script 設置
─────────────────────────────────────────────────────────────────
1. 建立 Google Sheet
2. 設置欄位: youtube_id | likes | youtube_link | facebook_link
3. 擴充功能 > Apps Script
4. 貼上 backend/Code.gs 內容
5. 更新 SHEET_ID 和 SHEET_NAME
6. 部署 > 新部署 > Web 應用程式 > 所有人可存取
7. 複製 Web App URL

📝 config.json 配置
─────────────────────────────────────────────────────────────────
{
  "api_base_url": "YOUR_WEB_APP_URL_HERE",  ⬅️ 必須更新！
  "site_title": "互動式影片活動",
  "max_likes_per_user": 10,
  "enable_like_limit": true
}

🔌 API 端點
─────────────────────────────────────────────────────────────────
GET  {api_base_url}?action=getVideos      # 獲取影片列表
POST {api_base_url}                       # 增加讚數
     Body: {"action":"addLike","youtube_id":"VIDEO_ID"}

🔍 故障排除
─────────────────────────────────────────────────────────────────
問題: API 請求失敗
解決: 檢查 config.json 中的 URL，確認 Web App 為「所有人可存取」

問題: 影片縮圖不顯示
解決: 確認 YouTube ID 正確，某些影片可能禁用嵌入

問題: 讚數不更新
解決: 檢查瀏覽器控制台（F12），驗證 Google Sheet 權限

問題: GitHub Pages 404
解決: 等待 5-10 分鐘，檢查 package.json 的 homepage 設定

問題: PowerShell 無法執行 npm
解決: 使用 cmd 或執行 Set-ExecutionPolicy RemoteSigned
